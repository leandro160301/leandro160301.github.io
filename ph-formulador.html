Descripción del Proyecto

Este programa está diseñado para trabajar con fórmulas de producción. Apto para todos aquellos procesos que requieran el uso de distintos ingredientes y varios pasos, dentro del proceso de producción. El indicador permitirá cargar todos los productos involucrados y luego confeccionar las distintas recetas, cargar turnos de trabajo, tolerancia de llenado, reportes varios, etc.

Tecnologías Utilizadas

  • Java
  • Android Studio
  • SQLite

Objetivo del Proyecto

El objetivo principal del proyecto es facilitar y optimizar el proceso de producción en industrias que requieren altos niveles de control y preparación de productos.

Proceso de Desarrollo

El proyecto se compone de tres etapas principales:

  • Parte estándar: Esta sección, descrita en detalle aquí, constituye la base del proyecto y contiene los componentes esenciales y comunes a todas las variantes del programa.
  • Variaciones de programas: Esta etapa incluye las diferentes versiones específicas del software, como la contadora, la embolsadora, el formulador, entre otras. Cada una de estas variantes está diseñada para cumplir con funciones especializadas dentro del ámbito industrial.
  • Sección de servicios: Esta última parte del proyecto se centra en la integración de servicios y está diseñada para facilitar la colaboración entre dos programadores. En este esquema, el primer programador se encarga del desarrollo de programas específicos (por ejemplo, una contadora), mientras que el segundo programador desarrolla la sección de servicios. Esta sección de servicios actúa como un "driver", abstrayendo los detalles de hardware y periféricos, como distintos protocolos de balanzas, modelos de impresoras y escáneres, entre otros. De esta manera, el primer programador puede concentrarse exclusivamente en la lógica del programa específico, sin preocuparse por las complejidades del hardware subyacente.

Esta estructura modular permite una mayor eficiencia y especialización en el desarrollo del software, optimizando tanto el tiempo como los recursos dedicados a la creación de soluciones industriales robustas y adaptables.

Conectividad

El equipo está diseñado para proporcionar diversas opciones de conectividad que permiten la integración con otros sistemas y dispositivos. Las opciones de conectividad incluyen:

  • Puertos serie: RS-232 y RS-485 para comunicación con dispositivos industriales.
  • Puertos USB para conexión con periféricos adicionales.
  • Conexión inalámbrica: Bluetooth clásico y BLE para comunicación sin cables.
  • Conectividad Wi-Fi para integración en redes locales y acceso remoto.
  • Puerto RJ-45 para conexión Ethernet y comunicación con redes cableadas.

Seguridad y Control

Para garantizar la integridad y seguridad de sus procesos, el PH-470 incluye un sistema de cuatro niveles de identificación: nivel 4 para programadores (personalización del fabricante), nivel 3 para administradores, nivel 2 para supervisores y nivel 1 para operadores. Esto restringe el acceso a modificaciones indeseadas, asegurando que solo el personal autorizado pueda realizar cambios críticos. Además, al acceder al equipo como operador, podemos realizar la producción sin comprometer los datos guardados en el sistema, quedando registrado todo lo hecho por cada operador correspondiente.
